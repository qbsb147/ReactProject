# React 영화 정보 프론트엔드

## 📌 프로젝트 소개

React를 활용하여 영화 정보를 보여주는 프론트엔드 애플리케이션을 개발한 개인 프로젝트입니다.
Zustand 기반 상태 관리, Axios를 이용한 백엔드 API 연동, styled-components로 다크/라이트 테마를 구현하며 UI/UX 향상과 컴포넌트 재사용성을 경험했습니다.

---

## 🛠 사용 기술

* React, JSX, CSS
* JavaScript ES6+
* Zustand (상태 관리)
* Axios (API 통신)
* styled-components (테마, 스타일링)

---

## 🏗 주요 기능

### ✔ 영화 목록 조회

* 페이징 처리 및 장르별 필터링
* Axios를 활용한 비동기 API 연동
* 영화 카드 컴포넌트로 목록 UI 구성

### ✔ 테마 기능

* 다크 / 라이트 모드 지원
* styled-components ThemeProvider 활용
* 사용자 경험을 고려한 UI 전환

### ✔ 상태 관리

* Zustand를 활용한 전역 상태 관리
* 로딩/에러 처리 및 페이지 상태 유지

## 🚀 실행 방법

1. 프로젝트 클론

   ```bash
   git clone [REPO_URL](https://github.com/qbsb147/ReactProject.git)
   ```
2. 의존성 설치

   ```bash
   npm install
   ```
3. 개발 서버 실행

   ```bash
   npm run dev
   ```
4. 브라우저에서 접속

   ```
   http://localhost:5173
   ```
